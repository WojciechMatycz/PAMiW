Podstawowe metody HTTP

Podstawowymi metodami HTTP zdefiniowanymi przez W3C(World Wide Web Consortium) są:

  • POST - Tworzenie zasóbu na serwerze
  • GET - Pobranie zasobu
  • PUT - Modyfikacja zasobu
  • DELETE - Usunięcie zasobu

Istnieją także inne metody HTTP jednak na razie przestawimy tylko 4 podstawowe
